Barusan saya pasang Google Analytics di portfolio ini.

Tidak ada alasan yang terlalu serius — saya hanya penasaran. Penasaran apakah ada yang mampir, dari mana asalnya, artikel mana yang dibaca, dan berapa lama mereka di sana.

Sekadar ingin tahu. Tidak lebih dari itu.

Tidak ada target traffic, tidak ada obsesi angka, tidak ada perubahan strategi konten berdasarkan data. Setidaknya untuk sekarang. Saya hanya ingin pasang, lalu lihat apa yang terjadi.

Semoga works.

## Satu Hal yang Hampir Terlewat

Karena portfolio ini di-deploy via GitHub Actions, Measurement ID dari Google Analytics tidak bisa langsung di-hardcode di kode — itu tidak aman untuk di-push ke repository publik.

Solusinya: simpan sebagai **secret variable** di GitHub repository settings, lalu panggil via environment variable saat build. Kalau kamu mau replikasi setup yang sama, jangan lupa langkah ini — tanpa secret yang terdaftar, analytics-nya tidak akan terbaca meski kodenya sudah benar.
